# Флаги битовой маски

Необходимо написать функцию `PrintCheckFlags`, которая принимает перечисление, 
представляющее собой битовую маску флагов для выполнений проверки, и выводит
на экран имена требуемых проверок.

Имена проверок должны быть выведены внутри `[]` через запятую без пробела. 

Если передан флаг отсутствия проверок, то необходимо вывести пустые `[]`.

Если переданное значение выходит из возможного диапазона значений, то вывод следует оставить пустым.
