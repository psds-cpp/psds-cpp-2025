# Двусвязный список (упрощенный)

Необходимо реализовать класс `SimpleList`, представляющий упрощенную реализацию
двусвязного списка `std::list` для строк `std::string` в динамической памяти. 

Каждый элемент списка (принято называть узлом `Node`) создается в динамической
памяти. Элемент содержит не только значение, но также указатели на следующий 
элемент и предыдущий (`next`, `prev`). Обычно узел выделяют в отдельную структуру.

Что хранить в самом списке? Список можно реализовать по-разныму. Один из вариантов
можно сделать список циклическим, тогда `next` последнего элемента будет ссылаться
на первый элемент, а `prev` первого на последний. Удобство такого подхода в том,
что указатели списка никогда не равняются `nullptr`, а, следовательно, не нужно
делать лишние проверки в коде. Также, часто вводят один фиктивный элемент, что
позволяет удобно представлять пустой список, а также для получения указателя
за последний элемент. Требования реализовывать конкретный вариант нет.

Также, чтобы метод получения размера списка работал не за линейное время O(n)
разумно добавить поле `size`. В стандартном контейнере, начиная с C++11, метод 
`size()` работает за константное время O(1).

Для удобства реализации также стоит подумать про написание приватных методов:

- Метод `Unlink` - принимает указатель на элемент и вывязывает его из списка,
  перевязывая указатели на следующего и предыдущего элемента друг на друга
- Метод `Link` - принимает новый узел и указатель на существующий элемент списка,
  после (или перед) которым необходимо привязать узел в список.

Класс предоставляет следующий функционал:

- Специальные методы
- Метод `Swap` - принимает другой список и меняет содержимое текущего списка с ним местами
- Метод `Size` - возвращает число элементов в списке
- Метод `Empty` - возвращает `true`, если список пуст
- Метод `PushBack` который вставляет элемент в конец списка.
- Метод `PopBack` - удаляет последний элемент списка. 
- Метод `PushFront` который вставляет элемент в начало списка.
- Метод `PopFront` - удаляет первый элемент списка.
- Метод `Back` - возвращает ссылку на значение последнего элемента.
- Метод `Front` - возвращает ссылку на значение первого элемента.
- Метод `Clear` - очистить список.

Методы вставки должны работать как с константными значениями, так и с временными.

Вставка и удаление элемента из списка при наличии указателя на элемент работает
за O(1).

## Примечание

- **Запрещено** использовать стандартные контейнеры
- Некоторые методы могут потребовать перегрузки для константного списка
- В данном случае рассматривается случай, когда можно создать значение по умолчанию,
  но в общем случае `std::list` может хранить элементы, которые не имеют конструктора
  по умолчанию. Как такое реализовать мы обсудим в будущих лекциях
- В данном случае реализация очень проста и далека от возможностей `std::list`, но 
  для большей функциональности необходимо знание итераторов
